VMI Setup

  1. Install deadsnakes ppa

    sudo add-apt-repository ppa:deadsnakes/ppa

  2. Install python3.7, virtualenv, git, distutils, redis

    sudo apt install -y python3.7 virtualenv git python3.7-distutils redis-server screen tmux

  3. Make folder structure

    cd ~ && mkdir VMI && cd VMI

  4. Clone Plan Repo

    git clone https://github.com/BluekaktusIT2022/Plan.git

  5. Clone Plan-UI Repo

    git clone https://github.com/BluekaktusIT2022/Plan-UI.git

  6. Get python3.7 path

    which python3.7

  7. Make virtualenv

    cd Plan && virtualenv -p <Step 6> bars

  8. Activate virtualenv bars

    source bars/bin/activate

  9. Switch to branch

    git checkout

  10. Install requirements

    python3.7 -m pip install -r requirements.pip

  11. Deal with errors

  12. Copy config

    cp api/sample_config.py api/config.py

  13. Export environment variables

    export FLASK_APP="api/wsgi.py" && export FLASK_ENV="development"

  14. Run Flask

    flask run

  15. Install Node Version Manager

    curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.38.0/install.sh | bash

  16. Export Stuff from output of <Step 14>

  17. Reload bashrc

    source ~/.bashrc

  18. Make sure nvm is installed

    nvm -v

  19. Install node

    nvm install v12.22.9

  20. Do npm install in Plan-UI folder

    npm i

  21. Do npm start

    npm start

  22. Install mongodb

    https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-ubuntu/

  23. Setup firewall
